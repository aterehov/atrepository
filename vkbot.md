# Название проекта

«Напоминания»

# Целевая аудитория

Данный продукт рассчитан на широкую аудиторию, поскольку число социальных
ролей, для выполнения которых необходимо не забывать о большом количестве
событий с определенным сроком выполнения, очень велико. Среди них можно
выделить такие, как:

·         Учащиеся старшей школы

·         Студенты

·         Преподаватели школ и ВУЗов

·         Программисты

·         И т.д.

# Проблемное поле

В наше время у многих людей в их жизни есть большое количество событий,
требующих от них конкретных действий в течение определенного срока. Часто они
накладываются друг на друга во времени, и приходится делать несколько дел
одновременно. Из-за большой занятости человек может забыть о некоторых задачах
и не выполнить их в срок. В результате этого для него, а если он работает в
команде, то и для его команды могут возникнуть неприятные последствия. У него
отсутствует средство, которое могло бы уведомить его о еще незавершенных
задачах. Бот ВКонтакте «Напоминания» позволит решить эту проблему, напоминая
пользователям о предстоящих событиях и их сроках.

Многие пользователи привыкли к работе с социальными сетями и пользуются ими
каждый день. Им будет удобно создавать напоминания, не выходя из чата
ВКонтакте.

# Функциональные требования

Бот ВКонтакте «Напоминания» предоставит пользователям следующие возможности:

·         Настройка часового пояса пользователя

·         Добавление и удаление напоминания

·         Просмотр списка напоминаний

·         Редактирование уже существующего напоминания

·         Отправка уведомления о предстоящем событии (созданного пользователем
ранее)

·         Повторная отправка уведомления

·         Возможность взаимодействия с ботом с помощью личных сообщений

·         Добавление напоминания для своих друзей

# Аппаратные требования

Серверная часть:

·         Виртуальный Linux-сервер с минимальными аппаратными требованиям (1-2
ядра, 512-1024Мб памяти) или облачные ресуры, например Heroku

Клиентская часть будет полностью работать посредством социальной сети
ВКонтакте , поэтому одно из перечисленного:

·         Браузеры: Google Chrome/Chromium, Mozilla Firefox, Microsoft Edge,
Safari/Safari Mobile, Opera, Yandex Browser

·         Приложение ВКонтакте

# Аналогичные продукты

·         Календарь Windows 10 – неудобен в использовании, для работы
одновременно на нескольких компьютерах требуется учетная запись Microsoft
(присутствует у сравнительно небольшого процента пользователей), часто
перестает запускаться без видимых причин

·         Календарь GNOME – работает только под Linux, охватывая таким образом
узкий круг пользователей, работает только на одном компьютере, требует
отдельной установки

·         Календарь Mail.ru – присылает уведомления только на электронную
почту

# Инструменты разработки

·         Python 3 – язык программирования

·         Visual Studio Code – среда разработки

·         vk_api – библиотека для выполнения запросов ВКонтакте на языке
программирования Python

# Этапы разработки

сценариев.

·         Написание системы хранения пользовательских данных.

·         Разработка механизма отправки напоминаний.

·         Запуск готовой версии бота.

·         Подготовка к защите.

# Возможные риски

·         Невозможность в необходимые сроки освоить API ВКонтакте из-за его
чрезмерной сложности

·         Трудности при работе бота одновременно с несколькими пользователями,
проблемы с пониманием принципа такой работы

·         Нехватка времени для реализации задуманного

·         Сложная бизнес-логика системы (разбор сообщений пользователя на
естественном языке, поддержание нескольких сценариев «бесед»).

·         Сложности с размещением бота на внешнем сервере

·         Возможны трудности с реализацией функции добавление напоминаний для
своих друзей
